Cycle: Ruby Enumerable Method

You should probably learn ctrl + C first.

Good thing I learned ctrl + C before I learned the method cycle

I tried this first:

a= ["a", "b", "c"]
a.cycle {|x| puts x}

Things got a little wild. Things I can't show here.

cycle enumerates like each, but starts over at the end. This causes an infinite loop, much like:

while (true) do
â€¦
end

Adding the (5) parameter to the cycle method will keep you from getting trapped in an infinite loop! Combining cycle with puts is like asking your copier to make 5 copies.

a= ["a", "b", "c"]
a.cycle(5) {|x| puts x}
