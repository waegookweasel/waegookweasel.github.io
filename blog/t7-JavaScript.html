Ruby Hashes vs. Java Script Objects

This is one way to create a new Ruby hash: 

 pants = { color: "green", size: 5 }

This is one way to create a new JavaScript object: 

var pants = {
    color: "green",
    size: 5,
    };

Notice how they both have keys and values. Additionally, Ruby hashes and Javascript objects are declared similarly with curly brackets.

Similary, you can add a new key and value pair to either.

Ruby: 

pants["price"] = "$29.00"

JavaScript: 

pants["price"] = "$29.00";
